# if-else
```
let color = "red";
if(color === "red"){
  console.log("red ball");
}else if(color === "blue"){
  console.log("blue ball");
}else {
  console.log("white ball")
}
```
# Falsy values
- false
- undefined
- null
- 0
- NaN
- the empty string ("")

# Switch

```
let color = "red";

switch(color){
    case "red":
        console.log("red ball");
        break;
    case "blue":
        console.log("blue ball");
        break;
    case "yellow":
        console.log("yellow ball");
        break;
    default:
        console.log("white ball");
}
```
